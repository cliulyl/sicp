## P226 - [练习 3.53]

``` Scheme
(define s (cons-stream 1 (add-streams s s)))
```

s 可以看成是 s 移动一个位置，跟自身相加。而第一个数是 1，第二个数就是 1 + 1 = 2，第三个数是 2 + 2 = 4 ....。

```
    1   2   4   8   16  32  64  ... = s 
+   1   2   4   8   16  32  64  ... = s
-------------------------------------------
1   2   4   8   16  32  64  128 ... = s
```

s 是 2 的幂 1、2、4、8、16、32、64、128 ...

运行程序可以验证结果。
